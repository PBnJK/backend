# Number Guessing Game
A fun little number guessing game!

Made for the
[roadmap.sh number guessing game](https://roadmap.sh/projects/number-guessing-game)
project.

## Running
### 1. Clone
```bash
git clone https://github.com/pbnjk/backend.git
```
### 2. Build
```bash
cd backend/number-game
go build
```
### 3. Run!
```bash
./number-game
